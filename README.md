## 🛠️ Tech Stack

### Frontend
- ⚛️ **React.js 19** — UI Library
- 🎨 **Tailwind CSS v4** — Styling
- ⚡ **Vite** — Build Tool
- 🔀 **React Router DOM** — Client-side Routing
- 📡 **Axios** — HTTP Requests
- 🎯 **React Icons** — Icon Library

### Backend
- 🟢 **Node.js** — Runtime
- 🚂 **Express.js** — Web Framework

---

## ✨ Features

- 🔐 User Authentication (Login / Logout)
- 🎬 Movie card slider with smooth scrolling
- 📱 Fully responsive design
- 🌐 Language selector (English / Hindi)
- 👁️ Password show/hide toggle
- 🔒 Protected routes (redirects to login if not authenticated)
- 🎨 Netflix-style UI with hero background
- ✉️ Email sign up section
- 🦶 Footer with links

## ⚙️ Getting Started

### Prerequisites
- Node.js installed
- npm installed

### Installation

**1. Clone the repo:**
```bash
git clone https://github.com/ruqayya29/netflix-clone.git
cd netflix-clone
```

**2. Install frontend dependencies:**
```bash
cd frontend
npm install
npm run dev
```

**3. Install backend dependencies:**
```bash
cd backend
npm install
node index.js
```
